# Apply

Read the tasks, spec, and design. For each assigned task:

1. Write a failing focused test when the behavior is testable.
2. Implement the smallest change that makes it pass.
3. Refactor while keeping tests green.
4. Record the completed task, test result, and any deviation.

Do not skip unassigned tasks or claim that a runtime-specific agent delegation occurred. Continue to
`click-sdd-verify` after the assigned batch is complete.
